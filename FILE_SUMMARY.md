## 💡 TIPS MENGGUNAKAN MATERI INI

### ✨ Golden Rules
1. **Jangan hanya membaca** - Execute setiap query!
2. **Practice regularly** - Minimal 1 jam per hari
3. **Modify queries** - Bereksperimen dengan kondisi berbeda
4. **Use EXPLAIN** - Understand query execution plan
5. **Build projects** - Apply knowledge ke project real

### 🎯 Best Practices
- Bookmark QUICK_REFERENCE untuk akses cepat
- Print CHEAT_SHEET untuk offline reference
- Create personal notes sambil belajar
- Share knowledge dengan peers
- Join online communities untuk help

### ⚠️ Common Mistakes to Avoid
- ❌ Using SELECT * (always specify columns)
- ❌ Function in WHERE (YEAR(date) = 2024)
- ❌ Subquery instead of JOIN
- ❌ Missing indexes on WHERE columns
- ❌ Not using prepared statements (SQL injection)
